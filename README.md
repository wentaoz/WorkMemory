# WorkMemory

## 中文简介

WorkMemory 是一款 macOS 本地工作记忆应用，用于记录会议、网页、文档、语音和待办上下文，支持隐私过滤、AI 摘要、行动项提取和按日期检索，帮助个人沉淀可回查的工作线索。

## English Overview

WorkMemory is a local macOS work-memory app for capturing meetings, web context, documents, dictation, and action items. It supports privacy filtering, AI summaries, action extraction, and date-based search for reviewable personal work context.
